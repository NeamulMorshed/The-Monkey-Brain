# Claude Code

Anthropic's agentic coding tool: a model that reasons about your code plus **built-in tools**
for file operations, search, execution, and web access. This page is the **hub** for the
Claude Code subject area in The Monkey Brain.

## The three things you configure
1. **What it does without asking** → [[permission-modes]] (and [[auto-mode]], [[plan-mode]],
   [[protected-paths]], [[permission-rules]], [[read-only-commands]], [[settings-precedence]]).
2. **What it knows / loads** → [[context-window]] (and [[compaction]], [[memory]]).
3. **How you extend it** → [[extend-claude-code|the extension layer]]: [[claude-md]],
   [[skills]], [[subagents]], [[agent-teams]], [[mcp]], [[hooks]] (+ the full
   [[hook-events|event catalog]]), [[code-intelligence]], [[plugins]].
