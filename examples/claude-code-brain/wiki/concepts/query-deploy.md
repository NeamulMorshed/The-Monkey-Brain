# Query (Deploy)

The "deploy" step of the [[knowledge-sdlc]]: answering questions *from the wiki* rather than
from scratch.

## Flow
1. Read [[index-and-log|index.md]] first to locate relevant pages, then drill in.
2. Synthesize an answer **with citations** to wiki pages (→ raw sources).
3. Choose the output form that fits: prose, comparison table, [[obsidian-ecosystem|Mermaid]]
   diagram, [[obsidian-ecosystem|Marp]] deck, chart.
4. **File good answers back.** A novel comparison, analysis, or discovered connection becomes
   a new [[syntheses]] page — indexed and logged — so explorations compound just like sources do.

> The key insight: good answers shouldn't disappear into chat history. Filing them back is
> what makes querying *additive* to the knowledge base.
