# Frontend Design (Claude Plugin)

> **Raw source:** [Frontend Design – Claude Plugin.md](../../raw-sources/Frontend%20Design%20%E2%80%93%20Claude%20Plugin.md) · **Origin:** claude.com/plugins · **Ingested:** 2026-06-17

## TL;DR
A Claude Code [[plugins|plugin]] that generates distinctive, production-grade frontends and
deliberately avoids generic AI aesthetics. Activates automatically when you ask Claude to
build UI.

## Key takeaways
- Establishes a **design framework before coding** — purpose, audience, and a specific
  aesthetic direction (brutalist, maximalist, retro-futuristic, luxury, playful…).
- Avoids clichés: generic system fonts, predictable purple gradients, cookie-cutter components.
- Focus areas: unexpected typography pairings, orchestrated/scroll-triggered motion, spatial
  composition with asymmetry, visual depth (gradients, textures, layers).
- Delivered as an auto-activating [[skills|skill]] — an example of the
  **[[plugins|plugin = packaged skill]]** pattern from [[extend-claude-code]].
